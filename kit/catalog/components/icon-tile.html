Icon Tile

A rounded brand-tinted square holding exactly one icon. Acts as a row leader, a button face, or a domain marker. Three sizes, four color variants.

Anatomy

A square with a hairline border, a faint colored fill, and a single Phosphor glyph centered. The fill and stroke both come from the same color token, with the fill at ~10% alpha for the tinted look.

  • --tile-size square side (default 36px). Drives width, height, and icon size.
  • --tile-bg fill, semantic color at 10% alpha.
  • --tile-fg icon color, the same semantic color at full strength.
  • Border 1px in matching color at 18% alpha (default: --c-hair).
  • Radius --r-md by default · --r-sm at sm · --r-lg at lg.

Colors

Four variants, default (neutral), amber, green, blue. The neutral tile uses ink on raised, no semantic accent.
