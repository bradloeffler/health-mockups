Pill

A small 24px capsule for trailing actions, status chips, and counts. Three semantic variants, amber (primary), green (success), ghost (neutral).

Anatomy

A 24px-tall horizontal capsule with eyebrow-spec text and optional leading glyph or dot. The shape is the same across variants; color and weight communicate state.

  • Height 24px · radius --r-pill · padding 0 --s-2.
  • Type eyebrow spec, mono uppercase, 10px, 700 weight, 0.16em tracking.
  • Gap --s-0 (4px) between glyph and label.
  • Icon size 12px when an icon is present.

Variants

Three semantic weights. Amber for the primary action on a card. Green for confirmed / success state. Ghost for low-emphasis labels and counts.
