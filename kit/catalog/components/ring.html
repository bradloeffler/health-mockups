Ring

Circular progress with an inner number and an outer label. Sized, colored, and filled by CSS custom properties, one component, many semantic uses.

Anatomy

An SVG with two stacked circles, track and bar, overlaid by an absolutely-positioned number, with a label flowing beneath as a separate text element. All sizing comes from --ring-size.

71
Recovery
  • --ring-size diameter of the ring. Drives both SVG and number positioning.
  • --ring-stroke line weight (default 6px). Auto-rescales to the viewBox.
  • --ring-color bar stroke. Pass a semantic token (--c-amber, --c-green, ...).
  • --ring-track track stroke (default --c-hair).
  • --ring-pct 0–100 fill percent. Drives stroke-dashoffset with a --m-slow transition.

Semantic colors

Color is meaning, not decoration. Green = recovery / good. Amber = workout / strain target. Blue = strain / load. Red = warning / low.
