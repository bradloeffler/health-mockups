Habit Row

A three-column row — leading icon · name · trailing element — separated by a hairline divider. Stacks inside a card to form a habit list, settings list, or any vertical menu.

Anatomy

A CSS grid with three tracks (auto 1fr auto). The middle track holds the name and absorbs whatever space the icon and trailing element leave behind. The bottom hairline disappears on the last row of a stack.

Vitamins AM
  • Grid auto 1fr auto · gap --s-3 · center-aligned.
  • Padding --s-2 vertical, 0 horizontal — relies on the parent for side padding.
  • Divider 1px --c-hair bottom; suppressed on :last-child.
  • Done state --done modifier strikes the name and dims the icon to 50%.

States

Two states, one modifier. Done struck-through plus icon dim communicates "complete" without removing the row.
