Progress

Slim horizontal progress bar — 4px tall, pill-rounded, color and fill driven by custom properties. Used in macro rows, daily-goal cards, and anywhere a linear fill reads better than a ring.

Anatomy

A full-width hairline track with a single fill drawn by an ::after pseudo. The fill width animates between values via a --m-slow transition.

  • --p-pct 0–100. Drives fill width (or >100 for overflow demo).
  • --p-color Fill color — pass a semantic token.
  • --p-height Track height (default 4px). Bump for emphasis.
  • Track --c-hair background, --r-pill radius.

Colors

Same three semantic colors as elsewhere. Amber = primary / goal. Green = protein / wins. Blue = strain / fat.

Fill states

Six states. 0% draws no fill. 100% fills the whole track. Overflow (>100%) clips at the right radius — useful when you've exceeded a target and want to communicate "done plus extra".
