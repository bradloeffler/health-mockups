Numeric

Standalone tabular number with optional unit suffix. Four sizes from sm to xl, xl is the marquee hero number. Always uses mono with tabular-nums so digit columns stay aligned.

Anatomy

Inline-flex container with baseline-aligned value and unit. The unit picks up the eyebrow spec, faint, mono, uppercase, narrow.

7,420steps
  • Container .k-numeric · inline-flex, gap --s-1, baseline aligned.
  • Value .k-numeric__val · mono, tabular-nums, --c-ink, weight --t-num-weight (300 at xl).
  • Unit .k-numeric__unit · eyebrow spec, --c-faint.
  • Sizes --sm 14 · --md 18 · --lg 22 · --xl 28.

Sizes

Four steps. xl drops to weight 300, a deliberate switch from semibold so the marquee number reads as quietly confident rather than shouty.

With & without unit

Unit is optional. Skip it when the surrounding label already names the dimension.

+ unit62ms
+ ratio unit5/ 8

Token-colored variants

Color is applied via inline color on the val span using a semantic token. The unit stays faint by design so the number leads.
