Metric & Metric List

A three-column grid (label · value · unit) that keeps numeric digits aligned across rows regardless of their width. Use it any time you stack two or more labeled numbers.

Anatomy

Each .k-metric uses display: contents to flatten its three children into the parent grid, so all rows share one set of column tracks. Digits are tabular and right-aligned in the middle column; that's what keeps the 1s, 10s, 100s columns lined up.

HRV 62 ms
Rest 7.4 hr
  • Columns auto auto auto, justified to end so the list hugs the right edge of available space.
  • Row gap --s-2 · column gap --s-2 · baseline aligned.
  • Value tabular-nums mono so 7.4 and 62 share digit-column widths.
  • Label & unit both use the eyebrow spec, mono, uppercase, faint, 10px.

Lengths

The list scales identically from 1 to N rows. Same column tracks, same alignment.
