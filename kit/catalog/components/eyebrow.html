Eyebrow

Tiny mono uppercase label. The locked typographic primitive for any label-shaped string — eyebrows, metric labels, unit suffixes, nav labels. Never restyle; always use the class.

Anatomy

A 10px mono string in faint ink, set in uppercase with wide tracking. Reads as metadata, never as content.

Recovery
  • Family --f-mono
  • Size --t-eyebrow-size (10px)
  • Weight --t-eyebrow-weight (700)
  • Tracking --t-eyebrow-track (0.16em)
  • Color --c-faint
  • Transform uppercase, line-height 1.2
