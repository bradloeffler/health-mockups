FORGE UI Kit

The component library FORGE is built from. Tokens, primitives, and composed patterns — everything authors need to assemble a screen without ever writing a raw hex or a raw pixel.

How to use

Two stylesheets, one icon registry. Compose primitives — never author raw values.

CSS 
<link rel="stylesheet" href="/kit/tokens.css" />
<link rel="stylesheet" href="/kit/kit.css" />Copy
Icons 
import { icons } from '/kit/icons.js';
// <i class="ph ${icons.recovery}"></i>Copy

Authoring rule: only token variables (--c-*, --s-*, --r-*, --t-*) appear in your markup. The harness lints any raw hex or raw pixel outside the kit and will fail the build.
